Serial Monitor
¿Cómo usar el simulador?
Escribe tu código usando las funciones setup() y loop() como en Arduino, pero en JavaScript.

Funciones y objetos disponibles:
  • setup(): Se ejecuta una vez al inicio.
  • loop(): Se ejecuta repetidamente (puede ser async para usar await delay(ms)).
  • Serial.begin(baud): Inicializa el monitor serie.
  • Serial.print(msg), Serial.println(msg): Imprime texto en el monitor serie.
  • await delay(ms): Pausa la ejecución por ms milisegundos (solo dentro de loop async).
  • pinMode(pin, modo): (placeholder, sin efecto en esta demo).
  • digitalRead(pin): (placeholder, siempre retorna 0).
  • analogWrite(pin, valor): (placeholder, sin efecto en esta demo).

Sobre el robot seguidor de líneas:
El robot simulado es un seguidor de líneas. Cuenta con sensores en la parte frontal que detectan si están sobre una línea negra o no. En una versión completa, podrías leer estos sensores usando digitalRead(pin) y controlar los motores con analogWrite(pin, valor).
  • Sensores de línea: Detectan si el robot está sobre la línea (negro) o fuera de ella (blanco). Usarías digitalRead(pin) para leerlos.
  • Motores: Se controlan ajustando la velocidad de cada rueda (izquierda/derecha) con analogWrite(pin, valor).
En esta demo básica, los sensores y motores son solo ejemplos y no están conectados a una simulación visual, pero puedes practicar la lógica de un seguidor de líneas usando las funciones disponibles.

Ejemplo básico:
function setup() {
  Serial.begin(9600);
}

async function loop() {
  Serial.println('¡Hola mundo!');
  await delay(1000);
}
